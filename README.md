# Library Management System

A desktop application developed with **C#**, **Windows Forms**, **SQL Server**, and **ADO.NET**.

## Features

- Add new books
- Update book information
- Delete books
- Search books
- Input validation
- Display data using DataGridView

## Technologies

- C#
- Windows Forms
- SQL Server
- ADO.NET
- Git

## Database

The application uses Microsoft SQL Server to store and manage book information.

## Project Structure

```
LibraryManagementSystem
├── Models
├── Data
├── Forms
├── App.config
└── Program.cs
```
